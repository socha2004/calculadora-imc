## Calculadora de IMC
Olá! Este projeto consiste uma página web que possui uma calculadora de IMC, dependendo de quanto você está pesando e medindo, será retornado um resultado explicado a situação atual do seu índice de massa corporal.

### O projeto foi criado com as seguintes ferramentas:
<ul> 
  <li>HTML5, para estruturação da página web</li>
  <li>CSS3, para estilização dos componentes visuais</li>
  <li>JavaScript, para programação da lógica de négocios e interatividade com usuários</li>
</ul>

### Como rodar o projeto?
É simples! Baixe o projeto  como pasta zip, extraia em qualquer lugar e abra o arquivo index.html
